It's on http://128.199.105.161/#/Chats if the server is still running.

## Project Setup
### Download Go and install
https://golang.org/dl/

### Install gorilla websocket
```
$ go get github.com/gorilla/websocket
```

### Go to the src directory and run
```
$ go run main.go
```
